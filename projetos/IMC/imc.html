Calculando meu IMC

O Índice de Massa Corporal é um cálculo para medir se a pessoa está ou não dentro do seu peso ideal, utilizando apenas a sua altura e peso. O cálculo é simples e ajuda o médico e/ou nutricionista a entender melhor o quadro do paciente.

É importante saber que este número é apenas uma base para tratamentos, não deve ser levado como verdade absoluta sem um acompanhamento médico. Caso ainda tenha dúvidas, procure sempre um especialista para auxiliar.
